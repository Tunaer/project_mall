# project_mall
基于Vue开发的移动端的电子商城

主要实现功能：
  商品首页界面、商品分类界面、商品详情界面、购物车界面


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
